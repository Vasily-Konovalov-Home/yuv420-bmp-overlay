# yuv420-bmp-overlay

Программа для наложения BMP изображения на видео в формате YUV420.

Программа читает BMP файл, преобразует его в цветовое пространство YUV420 по рекомендации BT.601 (студийный диапазон) и накладывает на каждый кадр входного YUV-потока. Поддерживается многопоточная обработка.

## Возможности

- Чтение несжатых BMP (24-бит RGB, без альфа-канала и палитры)
- Преобразование RGB в YUV420 с использованием коэффициентов BT.601 (ограниченный/студийный диапазон: Y 16-235, UV 16-240)
- Наложение полученной YUV-маски на все кадры исходного YUV-файла
- Настройка положения наложения (смещение по X и Y)
- Многопоточная конвертация (автоопределение числа потоков или ручное задание)
- Готовая сборка под Linux (GCC/Clang) и Windows (MinGW)
- В репозитории есть примеры BMP и сгенерированный HTML-файл для быстрой проверки

## Зависимости

- Компилятор с поддержкой C++11
- CMake версии 4.2 или выше
- Стандартная библиотека C++ (iostream, vector, thread, chrono)
- Внешних библиотек не требуется

## Сборка

### Linux (GCC или Clang)

```bash
git clone https://github.com/Vasily-Konovalov-Home/yuv420-bmp-overlay.git
cd yuv420-bmp-overlay
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
```

Или запустить: ./build_Linux.sh

###	Windows (MinGW)

Запустите пакетный файл build_Win.bat, который запросит путь к MinGW и скопирует необходимые runtime-библиотеки.

###	После сборки исполняемый файл yuv420_bmp_overlay (или yuv420_bmp_overlay.exe) появится в директории build/.

## Использование

yuv420_bmp_overlay <bmp> <input.yuv> <output.yuv> <width> <height> [offsetX] [offsetY] [numThreads]

### Параметры

- bmp - Путь к BMP-файлу (24-бит RGB, несжатый)
- input.yuv - Входной видеофайл в формате YUV420 (без заголовков, не .y4m!)
- output.yuv - Выходной файл (перезаписывается, если существует)
- width - Ширина видео в пикселях
- height - Высота видео в пикселях
- offsetX - Смещение BMP по горизонтали (по умолчанию 0)
- offsetY - Смещение BMP по вертикали (по умолчанию 0)
- numThreads - Число потоков для конвертации (0 = авто, по умолчанию)

## Пример

./yuv420_bmp_overlay logo.bmp video.yuv result.yuv 1920 1080 100 50 4

### Эта команда:

1. Считывает logo.bmp
2. Преобразует его в YUV420
3. Накладывает результат на каждый кадр video.yuv (размер 1920x1080) со смещением (100, 50)
4. Записывает обработанное видео в result.yuv
5. Использует 4 потока для ускорения конвертации

## Примечание

- Для маленьких картинок лучше использовать 1 поток, лишком много времени тратится на создание потоков
- Ширина и высота видео должны быть чётными. Если размеры BMP нечётные, программа автоматически обрежет их до ближайших чётных значений с предупреждением
- Если размер BMP превышает размер кадра, будет выдана ошибка
- Для просмотра YUV-файлов рекомендуется использовать yuvplayer
- Для конвертации видео в YUV рекомендуется использовать FFMPEG
  ($ ffmpeg -i video.mp4 -c:v rawvideo -pix_fmt yuv420p out.yuv)
